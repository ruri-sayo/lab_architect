# 素材・アセット管理マニュアル v2.0

本マニュアルは、フェーズ1〜5の実装完了に伴い、確定した必要素材リストと統合仕様をまとめたものです。

## 1. キャラクター（立ち絵）
| キャラクター | 表情差分 | ファイル名例 | 用途 |
|---|---|---|---|
| **秘書（栞里）** | 通常（微笑） | `shiori_normal.webp` | メイン画面、イントロ、通常会話 |
| | 落胆/悲しみ | `shiori_sad.webp` | 不正時、ゲームオーバー（逮捕）、厳しい報告 |
| | 驚き/焦り | `shiori_shock.webp` | 内部告発発生時、トラブル発生時 |
| | 暗黒微笑（デレ？）| `shiori_dark.webp` | 不正10回以上時の「共犯者」状態 |
